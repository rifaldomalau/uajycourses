Penjelasan

Array merupakan stuktur data yang dapat menyimpan sekumpulan data dalam satu variabel yang sama. Dengan adanya array memudahkan kita secara efisien untuk menyelesaikan masalah yang mempunyai banyak data. Berikut adalah konsep menyenai array: 1. Array dapat menyimpan lebih dari satu data dengan tipe data yang sama. 2. Ukuran dari array harus ditentukan saat pendeklarasiannya, sehingga perlu diketahui terlebih dahulu berapa maksimal data yang ingin ditampung di dalam variabel array. 3. Array bersifat static, artinya ukuran dari array tidak bisa berubah seiring berjalannya program. 4. Data pada array diakses menggunakan index, index adalah angka bilangan bulat dan index pada array dimulai dari 0 kemudian 1, 2 , 3 dan seterusnya.

‎

Visualisasi Array:

‎

• Array satu dimensi Array ini dideklarasikan satu dimensi dan menyimpan satu deret nilai yang bertipe sama. Contohnya menyimpan daftar nama, harga barang, dll. • Array multidimensi Array ini memiliki ukuran dimensi lebih dari satu dan array dua dimensi yang paling sering dipakai. Contohnya adalah membuat index posisi dll.

‎

Deklarasi Array:

Berikut struktur dari deklarasi variable array; tipeData namaVariabel[ukuran array]; atau tipeData namaVariabel[ukuran dimensi ke-1][ukuran dimensi ke-n];

‎

Gambar di atas dideklarasikan dengan 2 variabel yang mempunyai dimensi yang berbeda, variable array pertama merupakan array 1 dimensi dan array kedua adalah array dua dimensi. Ketika mendeklarasikan variabel array tentukan tipe data seperti variabel biasa, namun pada variabel array ditambahkan kurung siku yang di dalamnya berisi bilangan bulat yang menandakan banyaknya data yang akan disimpan.

‎

Gambar di atas merupakan cara manual dalam menginisialisasi array, pada cara ini data secara langsung diisi pada variabel array saat dideklarasikan, cara ini dapat digunakan Ketika data sudah diketahui serta harus diisi.

‎

Gambar diatas merupakan cara kedua dalam menginisialisasi menggunakan perulangan. Cara ini dapat digunakan Ketika data belum diketahui secara pasti atau data berasal dari inputan pengguna.

‎

Pengaksesan Array:

‎

Setelah variabel array dideklarasi, data ditampilkan dengan cara mengakses data pada variabel array dan yang dibutuhkan dalam mengaksesnya adalah index dari array tersebut, index dinyatakan di dalam kurung siku pada variabel array.

‎

Array Pada Prosedur Atau Fungsi:

Pada gambar di samping dibuat prosedur dengan nama initNama(string nama[]). Padaparameter prosedur atau fungsi, ukuran array tidak perlu diisi nilainnya (Kecuali untuk array 2 dimensi atau lebih). Dalam prosedur ini diinisialisasi array string yang diisi dengan ‘-‘ menggunakan perulangan for dimulai dari 0 sampai 4. Akses data array menggunakan index ke-i pada setiap data, nilai i yang dimulai dari 0 sampai 4 digunakan sesuai index yang dimiliki variabel array tersebut. Pada pemanggilan prosedur di main(), parameternya hanya perlu nama variabel arraynya saja
